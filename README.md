Method to run this:

Set up a Twilio WhatsApp number and obtain the API Key and Auth Token.

Set up and host medical_report.py on the GCP server.

Upload the list of medical reports using the medical-report_upload.py script.

Externally call the send-initial server endpoint hosted on the GCP server to initiate the process.
